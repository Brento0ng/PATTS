📡 ESP32 Auto-Registration

When a student presses the registration button on the ESP32 device:
1. ESP32 scans the barcode (Student Number)
2. HuskyLens captures and assigns a Face ID
3. ESP32 sends a POST request to http://YOUR_PC_IP:3000/api/register
4. Student is automatically registered and appears in the roster.

📡 ESP32 Auto-Violation Recording

        

          When a violation occurs via ESP32 device:

          1. ESP32 scans the barcode (Student Number)

          2. HuskyLens verifies the Face ID

          3. If both match, ESP32 sends a POST to http://YOUR_PC_IP:3000/api/violation

          4. Violation is automatically recorded in the database.
